# DjaVue

## Project setup
```
npm install
```

## Installing axios
```
npm install axios
```
## Installing bulma
```
npm install bulma
```


### Compiles and hot-reloads for development
```
cd ecomerece
npm run serve
```

## Installing Python Libraries
```
pip install requirements.txt
```

## Start the Back-end server
```
cd src
python3 manage.py runserver
```

### Compiles and minifies for production
```
npm run build
```
